## 🚀 &nbsp;Projets Phares

<details>
<summary>&nbsp;<b>🔐 &nbsp;SCANNER DE VULNÉRABILITÉS — Tool de Pentest</b>&nbsp;&nbsp;<code>Python</code> <code>C++</code> <code>Nmap</code></summary>
<br/>

> **Défi** : Développer un outil自动化 de scan de vulnérabilités pour identifier les failles courantes dans les infrastructures réseau.

Outil de sécurité personnalisé permettant l'automatisation de scans de vulnérabilités et la génération de rapports détaillés.

**Fonctionnalités :**
- 🔍 &nbsp;Scan de ports et services
- 🚨 &nbsp;Détection des **CVE** connues
- 📊 &nbsp;Génération de **rapports**
- ⚡ &nbsp;Intégration **Nmap** et scripts personnalisés

</details>

<br/>

<details>
<summary>&nbsp;<b>🛡️ &nbsp;SYSTÈME DE DÉTECTION D'INTRUSION (IDS)</b>&nbsp;&nbsp;<code>Python</code> <code>Scapy</code> <code>Wireshark</code></summary>
<br/>

> **Défi** : Concevoir un système de détection d'intrusion capable d'analyser le trafic réseau en temps réel et de détecter les comportements suspects.

Système IDS développé pour analyser les paquets réseau et identifier les tentatives d'attaque.

**Fonctionnalités :**
- 📡 &nbsp;Analyse de trafic **temps réel**
- 🔎 &nbsp;Détection des **scans de ports**
- 🚨 &nbsp;Alertes sur comportements suspects
- 📝 &nbsp;Logs détaillés au format **syslog**

</details>

<br/>

<details>
<summary>&nbsp;<b>🔑 &nbsp;AUTHENTIFICATION MULTI-FACTEURS (MFA)</b>&nbsp;&nbsp;<code>React</code> <code>Node.js</code> <code>TOTP</code></summary>
<br/>

> **Défi** : Implémenter un système d'authentification sécurisé avec génération de tokens TOTP.

Application d'authentification forte avec support TOTP et gestion de sessions sécurisées.

**Fonctionnalités :**
- 🔐 &nbsp;Génération de codes **TOTP**
- 🏠 &nbsp;Interface React sécurisée
- 🔒 &nbsp;Hashage de mots de passe (**bcrypt**)
- 📱 &nbsp;Support **QR Code** pour apps externes

</details>

<br/>

<details>
<summary>&nbsp;<b>🎮 &nbsp;BREAKOUT — Jeu C++ avec Raylib</b>&nbsp;&nbsp;<code>C++</code> <code>Raylib</code></summary>
<br/>

> **Défi** : Concevoir un jeu de A à Z en C++ bas niveau, avec une architecture propre et une vraie expérience joueur.

Jeu Breakout complet développé avec la bibliothèque Raylib. Le projet démontre une maîtrise des structures de données, de la gestion mémoire et de la boucle de jeu temps réel.

**Fonctionnalités :**
- 🗺️ &nbsp;**30 niveaux** uniques avec complexité croissante
- 🎵 &nbsp;Gestion du **son** et effets audio
- ⚙️ &nbsp;**Paramètres** personnalisables (difficulté, son)
- 🏠 &nbsp;**Menu principal** avec navigation fluide
- 🎯 &nbsp;**3 niveaux de difficulté** — Facile / Normal / Difficile
